GVC is an open-source project published by Moqsien.
You can star it or participate in development at https://github.com/moqsien/gvc.
It will be nice for you to do so to improve GVC.

GVC was born mainly for making things easy when you are demanding a nice dev environment for different programming languages.
It supports languages like Go, Python, Java, Rust, Nodejs, C/C++(Cygwin), etc. 
You can automatically download and install these compilers or interpreters through GVC with environment variables set correctly.
And multi-versions switching is also supported by GVC for Go, Python, Java, Nodejs.

GVC even can automatically download and install Visual Studio Code for you. And Neovim is also available. Likely, you would like to
try VSCode+Neovim to take advantage of both editors.

Besides, GVC can collect configuration files of VSCode and Neovim, backup these files to the WebDAV Server that you have configured.
So, you can easily rebuild the dev environment by using WebDAV when you have another machine or platform.

GVC itself also is highly configurable. You can find its configuration in "~/.gvc/backup/gvc-config.yml" or 
"C:\Users\Administrator\.gvc\backup\gvc-config.yml". Edit this configuration file, you will get different accelerations.

GVC产生的主要目的在于，当你需要不同语言的开发环境时，它能让你的工作变得更轻松容易。
它目前支持的语言有Go, Python, Java, Rust, Nodejs, C/C++(Cygwin)等。
你能通过GVC自动下载和安装这些语言的编译器或者解释器，同时它也能帮你正确地设置环境变量，而你自己几乎不用操心什么。
对于Go, Python, Java, Nodejs，GVC还提供了多版本自动切换功能。

GVC甚至能自动下载和安装微软的VSCode以及Neovim，这两者通过插件配合，应该你会喜欢。

还有，GVC能通过WebDAV(例如，坚果云盘支持WebDAV)备份VSCode和Neovim的配置文件，甚至VSCode的插件信息。
这样，你就能通过WebDAV轻松在不同的机器或平台上重建自己熟悉的开发环境。

另外，GVC自己是高度可配置的，你能在"~/.gvc/backup/gvc-config.yml"或者"C:\Users\Administrator\.gvc\backup\gvc-config.yml"找到
它的配置，可以根据自己的情况修改相关配置达到进一步加速的目的。

GVC还为国内用户提供了一键修改hosts文件的功能，用来加速github和vscode插件市场访问。

几乎所有的国内下载行为，都有国内源加速。
